# hscc

C Compiler 

First task is to implement a lexer using a finite state machine.


## Prerequisits 

Haskell compiler that is used: [GHC](https://www.haskell.org/ghc/)

```bash 
apt-get update && apt-get install ghc ghc-prof ghc-doc
```


## Run 

```bash 
ghc main.hs -o hscc && ./hscc 
```
